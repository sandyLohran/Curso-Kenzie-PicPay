HTML

Introdução

HTML (Hyper Text Markup Language) ou em Portugues Linguagem de Marcação de Hipertexto,
criado por Tim Berners-Lee Hoje em dia é considerado um componente base da web, permitindo
criação de websites e a inserção de novos conteúdos, como imagens e vídeos por meio dos Hipertextos.

Tags

As Tags são usadas para que o navegador entenda como as coisas deve ser apresentadas visualmente para o usuário.
As tags ficam dentro dos sinais, menor que (<) e maior que (>), usamos o caractere (/) para indicar fechamento da tag.

Exemplo

<tag> </tag>

Algumas tags

<h1> Titulo da Página </h1>
Vai do H1 até H6

<p> paragráfos </p>

Entidades

São caracteres reservados para o HTML como menor do que <, maior do que >, e comercial &, aspas "

Estrutura Básica para um pagina HTML

<!DOCTYPE html>
Essa tag/declaração informa qual a versão do HTML está sendo usada no site,
importante ele estar especificamente na primeira linha do seu arquivo HTML.

<html> </html>
Essa é a Tag raiz, todo o código deverá estar dentro dela.

<head> </head>
Contem descrições de sua página, como título <title>

<body> </body>
É o corpo da sua página, representa o conteudo do documento.

Listas no HTML

Possui dois tipo de lista:

  • UNORDERED LISTS
    Listas nao Ordenadas - A Ordem dos itens nao importa, podendo reorganizar a lista e continuara fazendo sentido
    Usamos a tag <ul> colocando os itens dentro <li> item </li> </ul>
  • ORDERED LISTS
    Listas Ordenadas - A ordem importa
    Usamos a tag <ol> colocando os itens dentro <li> item </li> </ol>

Incorporando Imagens

Para inserir uma imagem usamos o elemento <img> não possui tag de fechamento

COMANDO
